# Branding Skill - Ground Crew HQ

## Purpose
Remove leftover Lovable/default branding and make the app consistently branded as Ground Crew HQ.

## Required Files To Inspect
- index.html
- public/*
- src/pages/LaunchPortalPage.tsx
- src/components/AppLayout.tsx
- src/components/*
- package.json
- manifest / PWA files if present

## Brand Name
Primary: Ground Crew HQ
Short: HQ

## Requirements
- Browser title should be Ground Crew HQ
- Remove Lovable badge/icon references
- Remove Lovable favicon if present
- Use Ground Crew HQ or HQ in visible shell branding
- Keep professional green/grounds/facilities style
- Do not remove package dependencies unless confirmed unused

## Icon Direction
Use a simple HQ mark or grounds/facilities symbol.
Acceptable temporary icon:
- green square/circle with “HQ”
- no third-party branding

## Do Not
- Do not break Vite build.
- Do not remove app functionality.
- Do not change auth just for branding.

## Expected Result
No Lovable visual branding remains in the deployed app or browser tab.
